# Placeholders

Основной файл переменных: `/Users/admin/cursor/pencil-test/content/placeholders.json`.

Формат:

```json
{
  "[НАЗВАНИЕ СЕРВИСА]": "Reflow"
}
```

Правила:

- Меняйте только значения справа.
- Ключи (в квадратных скобках) оставляйте без изменений.
- Эти значения автоматически подставляются в `/legal`, `/legal/privacy`, а также в общий `header/footer`.

Реквизиты в `footer` тоже берутся отсюда:

- `[КОРОТКОЕ НАЗВАНИЕ ОРГАНИЗАЦИИ]`
- `[ИНН]`
- `[КПП]`
